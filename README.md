# SDK_WIN_x64_2025-0-634

Windows x64 SDK package (version `2025.0.634`) with binaries, headers, documentation, and sample projects.

## Repository Contents

- `binaries/` - prebuilt SDK binaries and libraries for Windows x64.
- `headers/` - C/C++ API header files.
- `documentation/` - generated HTML documentation.
- `samples/` - example projects and reference implementations.
- `getting_started.html` - local getting started guide.
- `migration_guide.html` - local migration guide.

## Quick Start

1. Open `getting_started.html` in a browser.
2. Configure your C/C++ project include path to `headers/`.
3. Configure your linker/library paths to the required files in `binaries/`.
4. Build and run one of the projects from `samples/` to verify setup.

## Notes

- This repository stores large binary artifacts.
- Keep directory structure unchanged if your build scripts reference relative paths.
